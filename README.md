📺 OLED Bitmap & Animation GeneratorAlat berasaskan web untuk menukar imej, GIF, dan video kepada kod tatasusunan (array) C++ yang serasi dengan skrin OLED SSD1306 (128x64). Sesuai untuk projek Arduino, ESP32, dan projek sistem terbenam (embedded) yang lain.✨ Ciri-Ciri UtamaSokongan Video & GIF: Ekstrak bingkai (frames) secara automatik untuk dijadikan animasi.4 Mod Dithering: Termasuk Floyd-Steinberg dan Bayer untuk kualiti visual terbaik pada skrin 1-bit.Kawalan Langsung: Laraskan brightness, contrast, threshold, dan scaling (Best Fit/Stretch/Fill) secara real-time.Output Automatik: Menghasilkan fail .h (data bitmap) dan .ino (kod Arduino) yang sedia untuk digunakan.🔌 Skema Pendawaian (Wiring)Projek ini dioptimumkan untuk skrin OLED I2C (SSD1306). Berikut adalah sambungan pin mengikut papan mikropengawal:1. ESP32 (DevKit V1)OLED PinESP32 PinVCC3.3VGNDGNDSCLGPIO 22SDAGPIO 212. Arduino UnoOLED PinArduino Uno PinVCC5V / 3.3VGNDGNDSCLA5SDAA4📚 Library DiperlukanSila pasang library berikut melalui Library Manager di Arduino IDE:Adafruit_SSD1306Adafruit_GFX_Library🛠️ Tutorial PenggunaanLangkah 1: Jana Kod di WebBuka fail index.html pada pelayar web.Muat naik fail gambar (.png/.jpg), GIF, atau video.Laraskan tetapan sehingga preview di bahagian OLED Live nampak jelas.Jika menggunakan video/GIF, tekan butang "Extract Smooth Animation".Klik butang Copy .h dan Copy .ino.Langkah 2: Sediakan Fail di Arduino IDEBuka Arduino IDE dan cipta sketch baru (Save as contohnya: OLED_Animasi).Cipta Fail Header:Klik pada ikon tiga titik (atau anak panah ke bawah) di penjuru kanan atas tab Arduino IDE.Pilih New Tab.Namakan fail sebagai VideoFrame.h (Pastikan ejaan sama seperti dalam kod).Paste (Tampal) kod yang disalin dari butang Copy .h ke dalam tab ini.Kod Utama (.ino):Pergi semula ke tab utama (fail .ino).Padam semua kod sedia ada dan Paste kod yang disalin dari butang Copy .ino.Langkah 3: UploadSambungkan ESP32 atau Arduino anda.Pilih Board dan Port yang betul.Tekan butang Upload dan lihat hasilnya di skrin OLED!📝 Nota TeknikalMemori: Animasi video menggunakan banyak memori Flash (PROGMEM). Untuk Arduino Uno, hadkan kepada 3-5 bingkai sahaja. Untuk ESP32, anda boleh memuatkan lebih banyak bingkai (sehingga 30-50 bingkai bergantung kepada saiz).Rotation: Jika skrin anda terbalik, anda boleh menggunakan butang Rotate di dalam web generator sebelum menyalin kod.Projek ini dibangunkan oleh miraiSedikit Tips Tambahan untuk Amirul:Untuk ESP32: Memandangkan kau banyak buat projek IoT (seperti AquaTrack), generator ini sangat sesuai untuk buat boot-up animation atau logo "Mirul Studio" masa skrin mula menyala.Kualiti Gambar: Gunakan preset "High Detail (Floyd-Steinberg)" untuk gambar potret atau pemandangan supaya nampak lebih realistik walaupun cuma hitam putih.
